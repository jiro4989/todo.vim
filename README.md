todo.vim
================================================================================
Vimでtodoメモを書くためのプラグイン

overview
--------------------------------------------------------------------------------
Markdown記法で使われる下記の記述に対して、マークのON/OFFをセットできます。  
マークのセットと同時に、セットした時の時間も入力します。

```markdown
- [ ] task1
- [x] task2
```

usage
--------------------------------------------------------------------------------
|キーマップ|動作|
|---|---|
|X|チェックのトグル切り替え|
|&lt;Leader&gt;tc|todo用のテキストファイルの作成と切り替え|

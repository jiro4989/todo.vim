todo.vim
================================================================================
Vimでtodoメモを書くためのプラグイン

overview
--------------------------------------------------------------------------------
Markdown記法で使われる下記の記述に対して、マークのON/OFFをセットできます。  
マークのセットと同時に、セットした時の時間も入力します。

```markdown
- [ ] task1
- [x] task2
```

usage
--------------------------------------------------------------------------------
上記文字列の上にカーソルが存在する状態で、デフォルトで X とタイプすると、下記の
テキストは以下のように変化します。

```markdown
- [ ] task1
↓
- [x]<2017/07/01 11:22:33> task1
```

逆に、その反対の実行も可能です。

```markdown
- [x]<2017/07/01 11:22:33> task1
↓
- [ ] task1
```
